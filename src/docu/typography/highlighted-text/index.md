## Highlighted Text

For highlighting text, you can use the following commands:

* For bold text
  * `<strong></strong>` tag
  * `<b></b>` tag
* For italic text
  * `<em></em>` tag
  * `<i></i>` tag
* For underline text
  * `<u></u>` tag
